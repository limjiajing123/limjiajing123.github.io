Robotic Arm

This is a group project aims to develop a versatile robotic arm equipped with an advanced end effector for precise object retrieval from shelves at varying positions. The arm utilises Robotis servo motors, enabling movement in a 3-dimensional spatial coordinate system.

In this project, I am primarily in-charge of the CAD design for the end-effector for my robotic arm and conducting usage of the robot model in Robotic Operating System(ROS) for simulation with the help of ROS moveit.

CAD Design for end-effector

The end-effector was designed with careful consideration of the limitations of the servo motors as well as for the task of collecting the items. The end-effector is equipped with a limit switch such that when an item is collect it will trigger the limit swith allowing us to know that the item has been collected.

Task of the robot arm is to collect item off the shelf

With the knowledge and use of inverse kinematics, the robot arm is able to move through a trajectory planned by sending angle values to the servo motor

Robot Arm in Rviz

Solidworks add-in called "SolidWorks to URDF Exporter" was used to allow for the convenient export of SW Parts and Assemblies into a URDF file. The exporter will create a ROS-like package that contains a directory for meshes, textures and robots (urdf files).

The MoveIt Setup Assistant is a graphical user interface for configuring any robot for use with MoveIt. Its primary function is generating a Semantic Robot Description Format (SRDF) file for your robot. Additionally, it generates other necessary configuration files for use with the MoveIt pipeline.

Robot Arm Simulation

The robot arm is set to move to a designated position and with the help of ROS MoveIt, the trajectory will be planned

For our robot arm task, the trajectory has been optimized to follow the most efficient and collision-resistant path.

The purpose of this simulation is for test run of the trajectory planning such that there will be minimal damaged to the real-life robot arm.
